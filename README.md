## Installation
```bash
git clone https://github.com/TheTrueJM/MovieReservationSystem.git
cd MovieReservationSystem
py -m venv .venv

# For Windows
source .venv/Scripts/activate
# For Linux / MacOS
source .venv/bin/activate

pip install -r requirements.txt
```
